# Понятный отчёт по прогону `final_experiment_2`

## Что это за файл

Этот файл сделан как краткая человеко-читаемая сводка.

Важно:

- `run_evaluation.json` в текущем проекте считает только качество карты высот и latency mapper-а;
- ошибки по траектории в стандартный evaluator сейчас не входят;
- поэтому здесь отдельно добавлен раздел по траектории.

## Короткий итог

- По маршруту прогон уже очень хороший, но формально остаются редкие выбросы выше `0.25 м`.
- По карте высот это сильный результат: средняя ошибка уже маленькая, а основная проблема теперь в редких крупных выбросах, которые тянут вверх `RMSE`, `p95` и максимум.
